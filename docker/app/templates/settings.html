Environment Variables

Configuration is managed through environment variables in your docker-compose.yaml or .env file. 

# Required - Claude Authentication
CLAUDE_CODE_OAUTH_TOKEN=your-oauth-token

# Optional - GitHub for private repos
GITHUB_TOKEN=ghp_xxxx

# Optional - Auto-clone repos on startup
CLONE_REPOS=https://github.com/user/repo1,https://github.com/user/repo2

# Optional - Web UI Authentication
AUTH_ENABLED=true
AUTH_USER=admin
AUTH_PASSWORD=your-password

# Optional - Graphiti Memory
GRAPHITI_ENABLED=true

After changing environment variables, restart the container for changes to take effect.
